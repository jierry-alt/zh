### ✨ 一句话介绍

一个让你彻底告别「文件传不了、链接挂了、速度龟速」的**永久直链托管工具**。

支持任意格式、单文件最高 200MB，永久储存 + 多线路镜像加速，白嫖党最爱！

### 🔥 核心亮点

- **永久有效**：上传一次，永久不失效
- **极速加速**：自动生成镜像加速链接（强烈推荐使用）
- **简单好用**：拖拽即传，一键复制
- **界面炫酷**：玻璃态设计 + 动态视频背景 + 背景音乐
- **手机友好**：完美适配手机端操作

### 🚀 使用方法

1. 打开 [Eternity 永久直链托管](https://jierry-alt.github.io/你的仓库名)
2. 点击或拖拽文件上传（单文件 ≤ 200MB）
3. 等待上传完成
4. 一键复制 **镜像加速链接**（速度更快更稳定）

**最常用场景**：
- 分享软件、配置、影视、音乐、游戏资源
- 提供订阅链接（小火箭、Clash、TVBox 等）
- 长期保存重要文件

### 📌 注意事项

- 单文件最大 200MB
- 不支持直接上传文件夹（可打包成压缩包后上传）
- 推荐使用生成的**镜像加速链接**
